Control Remoto por Infrarrojos (IR)

Aprende a recibir códigos IR desde tu control remoto (IR) para enviar órdenes a tu dispositivo.

Material requerido.
  • Tarjeta de Arduino o ESP, por ejemplo: Arduino Uno, Arduino Nano, Wemos D1 R2, Wemos D1 R32, NodeMCU, etc...
  • Una tarjeta multisensor o un receptor de infrarrojos, un control remoto IR, un LED RGB (rojo, verde y azul conectado a los pines D9, D11 y D10, respectivamente) y un zumbador conectado al pin D5.

Ejercicios Básicos
  1. Códigos IR

    Imprime los códigos infrarrojos recibidos en la consola para obtener información de cada uno de los botones que pulses.

    Pista:

    Usa la instrucción 'Código IR disponible' para verificar la disponibilidad de un código IR y luego, si está disponible, imprime el valor del código.

     
  2. Canales IR I

    Imprime el mensaje '¡Canal presionado!'al presionar los botones correspondientes a los números de canal del 0 al 9.

    Pista:

    Usa la instrucción 'Código IR disponible' para verificar la disponibilidad de un código IR y luego, si el número de código es menor que 9, imprime el mensaje.

     
  3. Canales IR II

    Ahora, mejora el ejercicio anterior para imprimir también un mensaje cuando se presiona el botón de apagado (código 12 en nuestro caso).

    Pista:

    Ahora, debes almacenar el código en una variable y extender la instrucción 'si' para verificar si el número de código es menor que 9 para imprimir el mensaje del canal o si es igual a 12 para imprimir un mensaje de apagado.

Ejercicios Intermedios
  1. IR 'Piano'

    Genera un tono de música basado en el código infrarrojo recibido para crear un piano controlado a distancia.

    Pista:

    Usa la instrucción 'Zumbador Avanzado' para convertir el número del código en una distancia de medio tono. Puedes establecer como frecuencia inicial 220Hz.

  2. Comandos IR

    Establece el color de un LED RGB según el código infrarrojo recibido. Si se presiona el botón de apagado, el LED RGB debe estar apagado, mientras que si se presiona '1', '2' o '3', se debe poner el LED RGB en rojo, verde y azul, respectivamente.

    Pista:

    Usa la instrucción 'Comando IR' para crear cuatro casos, con los códigos '12', '1', '2' y '3'. En cada caso, usa la instrucción 'RGB LED' para establecer el color correspondiente.
