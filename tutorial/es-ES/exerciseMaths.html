Matemáticas

Aprende a usar instrucciones matemáticas para realizar operaciones aritméticas, mínimo y máximo e incluso para generar una onda senoidal.

Material requerido.
  • Tarjeta de Arduino o ESP, por ejemplo: Arduino Uno, Arduino Nano, Wemos D1 R2, Wemos D1 R32, NodeMCU, etc...
  • Una tarjeta multisensor o un LED rojo conectado al pin D12, un potenciómetro conectado al pin A0 y un pulsador conectado al pin D2.

Ejercicios Básicos
  1. Multiplicar dos números

    Crea un simple programa que toma dos números y calcula el resultado de su multiplicación (los números son en representación Decimal y se obtienen a partir de los valores interpretados por el monitor serie). El resultado de la operación debe mostrarse por el monitor serie.

    Pista:

    Usa las instrucciones para interpretar un decimal y con la instrucción para realizar una operación aritmética calcula la multiplicación.

     
  2. Resto de una división

    Ahora calcula el resto de una división de dos números enteros (obtenidos desde el monitor serie, siendo el primer número introducido el dividendo y el segundo número el divisor).

    Pista:

    Usar la instrucción para el cálculo del 'modulo' con el símbolo '%' para calcular el resto de una división entre dos números. Debe siempre devolver un número entre 0 y N-1, siendo N el divisor.

     
  3. Distancia al origen

    Calcula la distancia al origen de un punto dado por sus coordenadas X e Y (introducido por el usuario a través del monitor serie).

    Pista:

    Usa la instrucción para interpretar números decimales para obtener los valores, después calcula el producto de cada uno de los números por sí mismo (es lo mismo que elevar al cuadrado) y suma el resultado de ambos productos y calcula su raíz cuadrada, siendo éste el valor a mostrar por el monitor serie.

     
  4. Escalado lineal

    Escala los valores leídos desde el potenciómetro, conectado al pin A0, de forma que su rango esté comprendido entre 0 y 100 y dibuja el valor para que se muestre por el Plotter de Arduino.

    Pista:

    Usa la instrucción 'Mapea' para realizar la operación de escalo lineal.

Ejercicios Intermedios
  1. Digitos Binarios

    Calcula el número de digitos necesarios para representar un número entero en su prepresentación binaria. Implementar una función que acepta como argumento de entrada el número entero y devuelve el número de dígitos. Usar, por ejemplo el número '123'.

    Pista:

    En un bucle Mientras, comprobar si el número es mayor o igual que la base en potencias de 2 (empezando por 2). Cada vez que eso sea cierto, el contador de dígitos debe aumentar (inicializado a 1).

     
  2. Minimo y máximo

    Usando las medidas obtenidas del potenciómetro (conectado al pin A0), calcular los valores máximo y mínimo de los mismos (los valores estarán escalados para estar entre 0 y 100). Al iniciar, enciende el LED rojo (conectado al pin D12) y realiza los cálculos para obtener el máximo y mínimo histórico hasta que se pulse el pulsador conectado al pin D2. Una vez pulsado, apagad el LED rojo y mostrar por consola los valores del máximo y mínimo de todas las medidas registradas.

    Pista:

    En un bucle Mientras, leer la señal analógica del pin A0 y convertir el valor en un número entre 0 y 100 usando la instrucción mapea. Usa las instrucciones min y max para calcular el valor mínimo y máximo obtenidos hasta el momento y almacena los resultados en dos variables diferentes (la variable que almacena el mínimo se inicializa a 100, mientras que la variable que almacena el máximo se inicializa a 0).

     
  3. Onda Senoidal

    Crea una onda senoidal con una frecuencia de 1Hz, una amplitud de √2, usando la instrucción 'senoidal'. Dibuja la señal usando el Serial Plotter de Arduino.
