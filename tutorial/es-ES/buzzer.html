Zumbador

Un zumbador, o también conocido como un beeper es un dispositivo de audio. Los usos típicos incluyen notificaciones, alarmas o dispositivos de confirmación.

Un zumbador pasivo tiene una capa piezoeléctrica que hace que la membrana vibre cuando se le aplica una corriente. Si se aplica una señal PWM, la frecuencia de dicha señal se convierte en un sonido con una vibración de esa frecuencia. Podemos generar notas con la frecuencia y duración deseadas.

Uno de los módulos más comunes es el KY-006 , un zumbador pasivo con tres pines, uno para conexión a masa, uno para la alimentación de 5 V y el tercero recibir una señal digital PWM que hace vibrar la membrana. Se pueden conectar a cualquier pin digital para comezar a hacer sonidos.

El zumbador también puede utilizarse para crear notas, que se pueden combinar para crear melodías. Puedes seleccionar la frecuencia deseada o la nota correspondiente dentro del bloque de Facilino. Debes de tener en cuenta la duración de las pausas que deseas reproducir.
Instrucciones básicas de sonidos con Facilino

Facilino proporciona varias instrucciones de sonidos y notas simples usando el zumbador. La primera instrucción permite generar notas basadas en sus nombres (puedes seleccionar la duración y decidir si deseas unirla con la siguiente). La segunda instrucción se usa para reproducir sonidos predefinidos, mientras que la tercera instrucción permite generar sonidos avanzados combinando algunos parámetros. La cuarta instrucción dejará de reproducir el sonido en el pin seleccionado y, por lo tanto, se podrá utilizar para otros fines.

Instrucciones de Melodias con Facilino

Facilino también proporciona instrucciones para reproducir melodías. Las melodías son de hecho una secuencia de frecuencias y duraciones de notas. La primera instrucción se utiliza para reproducir melodías. Cuando se selecciona la opción de tarea de fondo, la instrucción NO espera a que se finalice la melodía (reproduce la melodía en una tarea de fondo para permitirle ejecutar otras instrucciones en paralelo) . La segunda instrucción proporciona algunas melodías predefinidas, que pueden usarse en proyectos robóticos, por ejemplo. La tercera instrucción detiene la melodía si se está ejecutando en la tarea de fondo, mientras que la cuarta instrucción verifica si la melodía aún se está reproduciendo. Las instrucciones quinta y sexta son dos instrucciones representativas para crear melodías personalizadas en un pentagrama. Puedes añadir muchos tipos de notas y silencios para componer una melodía, similares a la quinta instrucción, pero asegúrate de añadir la sexta instrucción al final.
