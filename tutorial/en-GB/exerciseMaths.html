Maths

Learn how to use math instructions to perform arithmetic operations, minumum and maximum and even to generate a sinusoidal wave.

Required material.
  • An Arduino or ESP board, i.e.: Arduino Uno, Arduino Nano, Wemos D1 R2 or Wemos D1 R32, NodeMCU, etc...
  • A Multisensor shield or a red LED connected to pin D12, a pushbutton connected to pin D2, a potentiometer connected to A0.

Basic Exercises
  1. Multiply two numbers

    Create a simple program that takes two numbers (introduced by the user using the serial montior) and multiplies them (floating point operation). Resulting operation must be shown in the serial monitor.

    Hint:

    Use the parse decimal instructions to get the numbers within the arithmetic instruction to perform the multiplication.

     
  2. Reminder of a division

    Now compute the reminder of a divition of two integer numbers (use the serial monitor to introduce the numbers, being the first number introduced the dividend and the second one the divisor).

    Hint:

    Use the 'modulo' operation to compute the reminder of the division between the numbers. It must always return a value between 0 and N-1, begin N the divisor.

     
  3. Distance to the origin

    Compute the distance to the origin of a number given the coordinates X and Y (introduced by the user using the serial monitor).

    Hint:

    Use the parse decimal instructions to get the numbers and store them on floating point variables. Then, square the value of each coordinate and sum them up. Compute the squared root of the resulting operation and print the value to the serial monitor.

     
  4. Linear scaling

    Scale the values from the potentiometer connected to pin A0 so that their range is from 0 to 100 and plot the resulting scaled value on the Serial Plotter.

    Hint:

    Use the 'map' instruction to perform the scaling.

Intermediate Exercises
  1. Binary digits

    Compute the number of digits required to represent an integer number in a binary representation. Implement a function that accepts as input argument an integer number and returns the number of digits. Use, for instance number '123'.

    Hint:

    In a while loop, check if the number is greater or equal than the base 2 powers (starting from 2). Each time that this is true, a digits counter must be increased (initialized at 1).

     
  2. Minimum and maximum

    Using the potentiometer (connected to pin A0) measurements, compute the minimum and maximum values in volts (the value must be comprised between 0 and 100). At initialisation, turn on red LED (connected to pin D12) and perform the maximum and minimun computations until the switch SW1 (connected to pin D2) is pushed. Once pushed, turn off the red LED and show on the console the minimum and maximum values of all registered measurements.

    Hint:

    In a while loop, read the analog measurement in A0 and convert it into a number between 0 and 5 (using the map instruction). Use the min and max instructions to compute the corresponding minimum and maximum and save the results in two different variables (the variable storing the minimum value must be initialised with 100, while the variable for the maximum must be initialised with 0).

     
  3. Sinusoidal Wave

    Create a sinusoidal wave with 1Hz and amplitude of √2, using the presented instructions. And plot it in the Console with Serial Plot Data.
