Maths

Learn how to use math instructions to perform arithmetic operations, minumum and maximum and even to generate a sinusoidal wave.

All required Electronics are already integrated in the Multisensor Shield.
Required material.
  • Arduino Uno (and compatible) or Wemos D1R2/D1R32.
  • A Multisensor Shield.

Exercises
  1. Binary digits

    Compute the number of digits required to represent an integer number in a binary representation. Implement a function that accepts as input argument an integer number and returns the number of digits. Use, for instance number '123'.

    Hint:

    In a while loop, check if the number is greater or equal than the base 2 powers (starting from 2). Each time that this is true, a digits counter must be increased (initialized at 1).

  2. Minimum and maximum

    Using the potentiometer (connected to pin A0) measurements, compute the minimum and maximum values in volts (the value must be comprised between 0 and 5 volts). At initialisation, turn on red LED (connected to pin D12) and perform the maximum and minimun computations until the switch SW1 (connected to pin D2) is pushed. Once pushed, turn off the red LED and show on the console the minimum and maximum values of all registered measurements.

    Hint:

    In a while loop, read the analog measurement in A0 and convert it into a number between 0 and 5 (using the map instruction). Use the min and max instructions to compute the corresponding minimum and maximum and save the results in two different variables (the variable storing the minimum value must be initialised with 5, while the variable for the maximum must be initialised with 0).

  3. Sinusoidal Wave

    Create a sinusoidal wave with 1Hz and amplitude of √2, using the presented instructions. And plot it in the Console with Serial Plot Data.
