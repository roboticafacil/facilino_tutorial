Lyquid Crystal Display (LCD)

Learn how to print and scroll text to an LCD (I2C) screen with Facilino.

Required material.
  • Arduino Uno (and compatible) or Wemos D1R2/D1R32.
  • Arduino multisensor shield.
  • LCD 16x2 (I2C).

Exercises
  1. Static Text

    Write “This is a very long text” with a single instruction and then set the correct position so that words shown on each line are complete.

  2. Centered Text

    Write "This is a very" and "long text" in seperated lines (centered).

    Hint:

    Create two different strings that can be cast individual.

  3. Scrolling Text

    Print "Facilino" (centered) on the first row and then scroll a text on the second row print "This software is awesome".

    Hint:

    Predefined Function Blocks that scroll the text for you already exist, cast on two different areas of the screen.

  4. Setting Backlight

    Print “Facilino” and highlight the text by setting the backlight to LOW or HIGH.
