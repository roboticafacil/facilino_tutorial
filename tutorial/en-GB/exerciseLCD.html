Lyquid Crystal Display (LCD)

Learn how to print and scroll text to an LCD (I2C) screen with Facilino.

Required material.
  • An Arduino or ESP board, i.e.: Arduino Uno, Arduino Nano, Wemos D1 R2 or Wemos D1 R32, NodeMCU, etc...
  • A LCD Screen connected to I2C bus and a pushbutton connected to pin D2.

Basic Exercises
  1. Static Text I

    Write “This is a very long text” with a single instruction and the setup of the program.

    Hint:

    Use the 'Define LCD I2C (16x2)' and the 'LCD Print' instructions (do not check the set text position option).

     
  2. Static Text II

    Write “This is a very long text” with a single instruction as before. Then, set the correct position so that words shown on each line are complete.

    Hint:

    As before, use the 'LCD Print' instruction, but now in order to fit the text on the LCD Screen, print on the first line 'This is a very' and on the second line 'long text' (check the set text position option and set the correct rows for each text, begin 0 the first row and 1 the second row).

     
  3. Printing number of clicks

    Print on the LCD screen the number of times that we have pressed (short-click) the pushbutton connected to D2.

    Hint:

    Use the pushbutton instruction to detect the number of clicks. Create a global variable to count the number of clicks and increse the value each time a short-click is detected.

     
  4. Printing number of short-clicks and long-clicks

    Now, improve the previous exercise to print on the LCD screen the number of times that we have pressed the pushbutton connected to D2. Now, we have to distinguish between short-clicks and long-clicks, so that the number of clicks is printed on the first row and the number of long-clicks is printed on the second row.

    Hint:

    Use the pushbutton instruction to detect the number of short-clicks and long-clicks. Create two global variables to count the number of short-clicks/long-clicks and increse them each time a short-click/long-click is detected. You can reuse code if you create a procedure with four input arguments (two strings and two integers) to print the information on the LCD screen.

Intermediate Exercises
  1. Centered Text

    Write "This is a very" and "long text" in seperated lines (centered).

    Hint:

    Create two different strings that can be cast individual.

     
  2. Scrolling Text

    Print "Facilino" (centered) on the first row and then scroll a text on the second row print "This software is awesome".

    Hint:

    Print a static text on the first row as indicated, and on the second row, to scroll the text, use the 'LCD Scroll' instruction with a 100ms delay.

     
  3. Setting Backlight

    Print “Facilino” and highlight the text by setting the backlight to LOW or HIGH.

    Hint:

    Use, in the main loop, the 'LCD SetBacklight' instruction to set it to LOW or HIGH with a delay of 2000ms.
