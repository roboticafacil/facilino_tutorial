Arduino Boards

8-bit Boards

Arduino is an open-source hardware and software company, project and user community that designs and manufactures single-board microcontrollers. The Arduino platform has become quite popular with people just starting out with electronics. Unlike most previous programmable circuit boards, the Arduino does not need a separate piece of hardware (called a programmer) in order to load new code onto the board. Additionally, the Arduino IDE uses a simplified version of C++, making it easier to learn to program. Finally, Arduino provides a standard form factor that breaks out the functions of the micro-controller into a more accessible package.

Basic Arduino boards consist of an Atmel 8-bit AVR microcontroller with varying amounts of flash memory, pins and features. The boards use single or double-row pins or female headers that facilitate connections for programming and incorporation into other circuits. These may connect with add-on modules termed shields. Multiple and possibly stacked shields may be individually addressable via an serial bus. Most boards include a 5 V linear regulator and a 16 MHz crystal oscillator or ceramic resonator.

One of the most famouse board of Arduino is Arduino Uno, see the cloned version Arduino Uno (compatible), based on Atmega328P processor. It has 20 I/O pins, named from D0-D13 and A0-A5. Pins D0 and D1 are used for USB serial communication while pins A0-A5 can be used as analog inputs with a resolution upto 10-bits. PWM is implemented on pins D3, D5, D6, D9, D10 and D11. It has SPI (using pins D10-D13) and I2C communication (using pins A4-A5). External interrupts are implemented on pins D2 and D3.

An alternative to Arduino Uno boards is to use Arduino Nano (compatible), together with a Arduino Nano Extension Shield. The main advantage of this configuration is that it allows you to have access to more power and ground pins. This can be easily combined with many of the electronics described in Electronics Section.

32-bit Boards

The new generation of Arduino boards use a SAMD21 32-bit ARM MCU, again with varying amounts of flash memory, pins and features. However, the one of the main differences with respect to previous boards is their interconnectivity capability, which make them very suitable for IoT applications. Among these boards, probably, Arduino Zero is one of the most remarkable boards, which includes 20 I/O pins, 6 analog pins (A0-A6) with 12-bit resolution and one analog output (10-bit resolution). All pins can be used as PWM, but pins D2 and D7. It has two UARTs (one used for serial USB communication), SPI and I2C. Another important characteristic of the new generation of boards is that they work at 3.3V.
