Light Dependent Resistor (LDR)

Learn how to read raw and calibrated values from an LDR sensor.

Required material.
  • An Arduino or ESP board, i.e.: Arduino Uno, Arduino Nano, Wemos D1 R2 or Wemos D1 R32, NodeMCU, etc...
  • A LDR light sensor connected to A1 pin and a RGB LED (red, green and blue connected to pins D9, D11 and D10, respectively).

Basic Exercises
  1. Measuring Light I

    Print values for raw measurements from a LDR connected to pin A1.

    Hint:

    Print the analog value of the signal connected to A1 corresponding to the LDR. Include a 100ms delay.

     
  2. Measuring Light II

    Print "too much light" or "too dark" if there is too much light or if it is too dark, where 'too much light' is the case where the read value is larger than 1000 (Wemos D1 R32) or 250 (Arduino Uno) and 'too dark' it is the case where the read value is smaller than 600 (Wemos D1 R32) or 150 (Arduino Uno).

    Hint:

    Now, store the read value on a variable and with an 'if' instruction, check if, either, 'too much light' condition or 'too dark' condition occurs ( values might be different depending on your lighting conditions).

     
  3. Measuring Light III

    Now, following with the ideas of the previous exercise, set a white color on the RGB LED (connected to pins D9, D10 and D11) when it is too dark and switch off (set the RGB LED to black) when there's too much light.

    Hint:

    Now, instead of printing messages, you need to set the pins D9, D10 and D11 to HIGH or LOW depending on the light conditions.

     
  4. LED Control

    Now, based on previous exercises, set the RGB LED light intensity proportional to the measured light.

    Hint:

    Use the 'Advanced Map' instruction to generate a linear scaling between the measured light and the PWM signal to be applied on to each of the RGB LED channels.

Intermediate Exercises
  1. Calibrated Measurement I

    Print values for raw measurements and calibrated measurements in Percent and Lux.

    Hint:

    The sensor just sends an analog value to the microcontroller, we now have to convert that analog signal to Luminance linear measured in the unit percent or logarithmic measured in the unit Lux using the calibrated instructions.

     
  2. Calibrated Measurement II

    Now, based on previous exercise, set the minimum and maximum Lux values so that the percent values now are properly adjusted.

    Hint:

    You need to use the 'Set Max. Lux' and 'Set Min. Lux' instructions to set the corresponding values in the 'Setup' section.

     
  3. Calibrated LED Control

    Now, based on previous exercise, set the minimum and maximum Lux values so that the percent values now are properly adjusted and used to control the RGB LED light intensity as in Exercise 4.

    Hint:

    You need to store the calibrated luminance value (percent) in a variable and map it into another variable so that 100% is 0 and 0% is 255.
