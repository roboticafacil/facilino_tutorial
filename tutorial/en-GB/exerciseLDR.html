Light Dependent Resistor (LDR)

Learn how to read raw and calibrated values from an LDR sensor.

Required material.
  • Arduino Uno (and compatible) or Wemos D1R2/D1R32.
  • Arduino multisensor shield.
Worked instructions
Exercises
  1. Measuring Light.

    First print values for raw measurements, to get values to be defined as "too light" and "too dark". Print messages if there is too much light or if it is too dark. 

  2. Calibrated Measurement.

    Print values for raw measurements and calbrated measurements in Percent and Lux.

    Hint:

    The sensor just sends an analog value to the microcontroller, we now have to convert that analog signal to Luminance linear measured in the unit percent or logarithmic measured in the unit Lux.
