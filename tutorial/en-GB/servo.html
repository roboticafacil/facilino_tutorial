Standard Servo

A Servo Motor is rotary actuator, that allows for more precise control of angular or linear position, velocity and acceleration. It is a specialised position controlled DC Motor with a small driver.

A Servo includes a DC Motor, gearbox, IC driver or control circuit and potentiometer. The accuracy of simple servo motors is, in general, not very good, but sufficient for most applications. You can find them of different sizes and torques but it's connection and control is, somehow, standarised. For small torque and low-cost applications, SG90 servo is one of the most widely used options.

A servo is controlled with an PPM signal, a 20ms periodic signal with a variable pulse width. If the time of the pulse width is about 0.5ms, then the servo moves to 0 degrees. If the pulse width is 1.5ms then the servo is at the centered position of 90 degrees. If the pulse width is 2.5ms then the position is 180 degrees. Due to the simplicity of the signal, it is implemented in most of microcontrollers using a PWM (pulse width modulation).

Continuous Rotation Servo

A Continuous Rotation Servo Motor is very similar, in aspect, to a standard servo. Ineed, they can be controlled using a PPM signal too, but the difference is that you can control the speed of a continuous rotation servo, but not the position. Indeed the electronics are very similar, but one of the most remarkable differences is that the potentiometer is not attached to the motor's shaft, but adjustable from outside the servo case. This potentiometer is used to set the angle at no speed. If the angle is set to 90 degrees, this means to be stopped, while if the angle is either 0 degrees or 180 degrees, the servo will start moving a maximum speed (0 degrees means clockwise and 180 degrees means counter-clockwise). For small torque and low-cost applications, FS90R servo is one of the most widely used options.

Coding with Facilino

Facilino provides instructions to control the position of a standard servo and to control the speed of a continuous rotation servo. The first instruction sets the position of a standard servo. The input should be a value between 0 and 180 degrees. The second instruction sets the speed of a continuous rotation servo a value between -100% to 100%.
