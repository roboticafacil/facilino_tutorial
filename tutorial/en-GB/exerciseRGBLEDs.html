RGB LEDs

Learn how to set the color of a RGB LED and dimm the light using PWM (analog write).

Required material.
  • An Arduino or ESP board, i.e.: Arduino Uno, Arduino Nano, Wemos D1 R2 or Wemos D1 R32, NodeMCU, etc...
  • A Multisensor shield or a RGB LED (red, green and blue connected to pins D9, D11 and D10, respectively).
RGB Colors
Colors can be represented in multiple ways. One of the most common representation is the RGB coding that corresponds to a color as a result of combining red (R), green (G) and blue (B) light. Depending on the combination of these colors, we can generate a wide variety of them. With a resolution of 8 bits per channel, corresponding to 256 levels of color intensity in each of the channels, we can generate up to 2563 = 16,777,216 color combinations. In most graphic editing programs, you can select a color through a color palette to get the RGB code of that color. You can also access here to select/generate different types of colors (it also has tables with the most common colors).
Rainbow colors
VIBGYOR is an acronym for Violet, Indigo, Blue, Green, Yellow, Orange and Red. These are the seven colors displayed on a rainbow and frequently referred as VIBGYOR. The following table shows these rainbow colors and their corresponding RGB values.

Basic Exercises
  1. Set LEDs individually

    At start up, turn on and turn off each of the LEDs connected to the RGB LED using a standard digital write instruction.

    Hint:

    Use the 'DigitalWrite' instruction and set the state to HIGH or LOW on each of the LEDs.

     
  2. More colors

    Now, create more colors by combining the channels. For instance, you can create a Yellow color by turning on red and green. Create a new program in which you create yellow (red and green), violet (red and blue) and cyan (green and blue)

    Hint:

    Use again the 'DigitalWrite' instruction, but now you need to turn on red and green LEDs at the beginning and turn on and turn off the corresponding channel to generate the other colors. Please, take into account that green is connected to D11 and blue is connected to D10.

Intermediate Exercises
  1. Light Dimm

    Regulate the light intensity of the Red LED, so it smoothly changes from 0 to full intensity in a loop. Repeat the procedure for the other colors too.

    Hint:

    Use 'AnalogWrite' instruction from 0 to 255 on the red channel.

     
  2. Red to Yellow color fade

    Implement a function that performs a color fade, from one color to another. For simplicity, we first start by fading two channels, red and green to perform a fading between red (R=250,G=0) to yellow (R=255,G=255) and then fades from yellow to red. The function performs a loop of 100 iterations progresively changing from the initial color of red and green values to the final ones with a given delay of 10ms.

    Hint:

    Define a function and add five input arguments: 'Red1', 'Green1', 'Red2', 'Green2' and 'time'. The 'time' argument is of type 'int', while the rest of arguments are of type 'byte'. Compute the increment on each iteration as: deltaR=(float)(Red2-Red1)/100 and deltaG=(float)(Green2-Green1)/100, being 'deltaR' and 'deltaG' float variables and the difference between the colors is casted into a float number to avoid loosing precision using 'To number' instruction as indicated with the '(float)' cast operator. Then in a 'Count with' for loop, compute the corresponding value for the 'Red' and 'Green' channels as: Red=(byte)(deltaR*i+Red1) and Green=(byte)(deltaG*i+Green1), being i the iteration from 1 to 100 of the for loop. Set the dutycicle of each channel and wait the indicated amount of time.

     
  3. Rainbow fade

    Now, extend the previous exercise, so that fades three channels: red, green and blue. Generate the rainbow colors with a fading effect.

    Hint:

    Add more input arguments to include 'Blue1' and 'Blue2' and compute the corresponding increment and blue color using similar formulas as before.
