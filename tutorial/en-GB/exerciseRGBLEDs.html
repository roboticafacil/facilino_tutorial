RGB LEDs

Learn how to set the color of a RGB LED and dimm the light using PWM (analog write).

Required material.
  • Arduino Uno (and compatible) or Wemos D1R2/D1R32.
  • Arduino multisensor shield.
  • One RGB LED.

Exercises
  1. Alternate Colors

    Using the RBG LED, alternate Red, Green and Blue colors at maximum intensity. Let them swap continous between those colors with a little break in between.

  2. Random Color

    Generate a random color using the RGB LED every second.

    Hint:

    Use the 'random' instruction to create a value between 0 and 255 for each LED.

  3. Light Dimm

    Regulate the light intensity of the Red LED, so it smoothly changes from 0 to full intensity in a loop. Repeat the procedure for the other colors too. Afterward regulate all of them together, to create a beautiful rainbow fade effect.

    Hint:

    Use different timing for the LEDs to create a better more colorful effect.
