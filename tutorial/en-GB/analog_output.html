Analog Output (PWM)

An Arduino's analog outputs detailed description

Analog signal

An analog signal is a signal that can take any value between 0V and 5V (for Arduino Uno). Therefore, the main difference with respect to a digital signal is that digital signals can only take two possible values, 0V or 5V, while an analog signal could be any real number between 0V and 5V, for example 1.2V.

(From Wikipedia) An analog signal is any continuous signal for which the time varying feature (variable) of the signal is a representation of some other time varying quantity, i.e., analogous to another time varying signal.
Analog Outputs

Creating accurate precise analog signals can be a challenge for most of microcontrollers. Instead of using a Digital-to-Analog converter (DAC), they commonly use a method called Pulse Width Modulation or PWM.

PWM, is a technique for getting analog results with digital means. Digital control is used to create a square wave, a signal switched between HIGH (5V) and LOW (0V). This on-off pattern can simulate analog continous voltages in between full 5V and 0V by changing the portion of the time the signal spends on versus the time that the signal spends off. The duration of "on time" is called the pulse width. To get varying analog values, you change, or modulate, that pulse width. If you repeat this on-off pattern fast enough with an LED for example, the result is as if the signal is a steady voltage between 0 and 5v controlling the brightness of the LED.

In the graphic below, the green lines represent a regular time period. This duration or period is the inverse of the PWM frequency. In other words, with a microcontroller PWM frequency at about 500Hz, the green lines would measure 2 milliseconds each. The black line is the PWM signal, switching at the indicated frequency. The ductycicle is the percentage that the signal stays HIGH with respect to the time that stays LOW. The mean value of that signal is the red line, which is the analog signal seen by many devices.

Coding with Facilino

In order to set the value of an analog output in Facilino we need to use the following instructions. The first instruction requires a digital pin with PWM capabilities and the dutycicle, a number between 0 and 255 for a 8-bit resolution. When the dutycicle is 0 the signal stays at 0V and when the dutycicle is 255 the signal constant stays at 5V. Any other value in between will set the width of the ON time of the PWM signal. Facilino instruction for ESP processors allows to set the PWM frequency and the resolution. The second instruction returns a valid PWM pin.
