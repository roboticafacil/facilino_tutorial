Relay and Servo Actuators

Learn how to control with a relay and move a standard servo.

Required material.
  • An Arduino or ESP board, i.e.: Arduino Uno, Arduino Nano, Wemos D1 R2 or Wemos D1 R32, NodeMCU, etc...
  • A Multisensor shield and a Relay module connected to pin D7 and a SG90 servo motor connected to pin D8. If you don't use the multisensor shield, you need a pushbutton connected to pin D2 and a potentiometerconnected to pin A0.

Basic Exercises
  1. Fan Control with a Relay

    With a relay connected to pin D7 and a pushbutton connected to pin D2, activate the relay each time you press the pushbutton. You can control a PC fan with a relay so that every time you push the button, the fan will start spinning.

    Hint:

    Use the 'Pushbutton' instruction so that every time it is on a pressed state, you set the relay to HIGH, otherwise, you set the relay to LOW.

     
  2. Fan Control with a Relay and Potentiometer

    With a relay connected to pin D7 and a potentiometer connected to pin A0, activate the relay if the potentiometer value is above 800 (in Arduino) or 3200 (in ESP).

    Hint:

    Use an 'if' instruction to detect the case where the potentiometer value is above 800 (or 3200).

     
  3. Moving a servo

    Move a servo to its 0º position and 3000ms move it to its 180º and back again to 0º after 3000ms. You can attach a blade waved by the servo to produce a air flow similar to a fan waved by hand.

    Hint:

    Use the servo instruction to set it to 0º or 180º and the proper delays in the main loop.

     
  4. Servo Sweep

    Make a servo sweep of all possible positions between 0º and 180º with a 10º step (increment) each 500ms. Once the last position is reached, move back to 0º in a rapid movement of just 500ms.

    Hint:

    Use a 'for' instruction to loop between 0 and 18. On each iteration set the servo position based on the loop iteration times 10. Once the loop is done, then move to 0º.

Intermediate Exercises
  1. Temperature Control with a Relay

    Now, following the ideas of exercise 2, use a temperature sensor DHT11 connected to pin D4 and if the temperature is above 25ºC, then activate the relay, which connects a fan to decrease the temperature.

    Hint:

    Replace the 'AnalogRead' instruction of a 'Temperature Read' one. You need to set proper temperature conditions for this example to work.

     
  2. Servo Wave

    Generate a sinusoidal wave with a servo so that the position varies between 45º and 135º (this might be helpful for walking patterns in some robots or for generating a air flow to decrease temperature with a blade attache to the servo emulating a fan waved in hand).

    Hint:

    Use the 'Sinusoid' instruction so that the amplitud is 45, the frequency should be slow, say 0.1Hz and the offset 90º (the phase is not that relevant in this case).
