Sense temperature with high precision

LM35 Sensor

The LM35 is an analog temperature Sensor. It has a linear response with a resolution of 10mV per Celsius Degree. Around 25°C it has the highest Precision with a accuracy of 0.5°C, its range is -55°C to 150°C. It has three Pins, one connected to Ground, one connected to 5V Supply and the last one for the analog signal. LM35 sensor can be found as one of the included sensors in the Multisensor Shield

KY-070 temperature sensor module provide a breakout board to start practicing with this sensor. You can connect its analog signal to a microcontroller analog input to measure the temperature.

Coding Melodies with Facilino

Facilino includes an instruction for reading the temperature of a LM35 sensor. The instruction accepts as input the analog pin number and returns the temperature in Celsius Degrees.
