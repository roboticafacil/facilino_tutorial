Potentiometer

A Potentiometer is an adjustable variable resistor. It has three pins and usually a turn-dial, its position determines the resistance value and therefore the output voltage. The potentiometer essentially functions as a variable voltage divider.

It is used to create a manual analog signal. Two pins connect to each side of a regular resistor and the third (usually the middle pin) connects to a movable part, wiper. This wiper is connected to the middle of the resistive material. Relative to its position this connection has a linear progressing resistance.

The side pins are connected to 5V and Ground (0V). The middle pin is connected to an analog pin. The connection between 5V to Ground and the conncetion between 5V and analog act as a voltage divider. You will now receive a Analog Signal relative to the angular or linear position between 0V and 5V.

Joysticks

Potentiometers operated by a mechanism can be used as position transducers, for example, in a joystick. The most simplified joysticks used Arduino applications is 2-Axis Joystick module KY-023 are very similar to the ones used in PS2 remote, providing an X-axis analog position and Y-axis analog position. Coding with Facilino

Facilino provides does not provide specific instructions for general purpose potentiometers. However, it is very common to use the first instruction to scale the value read from the analog input into something meaningful such a position percentage (being the first input the read analog input value and the second input the constant number 100). The second and the third instructions are used in Facilino to read the magnitude (distance to the center) and direction (angle) of a 2D analog joystick device.
