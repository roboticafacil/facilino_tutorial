DHT11 Sensor

Humidity and Temperature can be measured with a varying array of sensors. One of the most common and easy to use is the DHT11. It measures a Humidity provided in percentage (unit precission) and a Temperature provide in Celsius with degree precision.

It uses one digital signal for transmitting and receiving data. The DHT11 has four pins; one ground, one 5V voltage supply one sending data and one free pin.

KY-015 is a module that includes a pull-up resistor to interface with the sensor using one single digital pin.

Coding with Facilino

Facilino provides several instructions to work with DHT sensors. The first instruction returns the temperature in Celcius degrees, while the second instruction returns the relative humidity. The third and fourth instructions differentiate between high, confortable and low temperature or humidity (human confortable). Both instructions use temperature and humidity to determine whether or not it is whithin confortable ranges. You must wait, at least 1000ms, between two consecutive measurements
