Infrared Remote (IR)

Learn how to receive IR codes from your IR remote controller to command 'orders' to your device.

Required material.
  • An Arduino or ESP board, i.e.: Arduino Uno, Arduino Nano, Wemos D1 R2 or Wemos D1 R32, NodeMCU, etc...
  • A Multisensor shield or an infrared receiver, an IR remote, a RGB LED (red, green and blue connected to pins D9, D11 and D10, respectively) and a buzzer connected to pin D5.

Basic Exercises
  1. IR Codes

    Print the received InfraRed Codes on the console to get an overview of the communications channel.

    Hint:

    Use the the 'IR Code available' instruction to check the availability of a IR code and then, if available, print the code value.

     
  2. IR Channels I

    Print the message 'Channel pressed!" when pushing the buttons corresponding to the channel numbers from 0 to 9.

    Hint:

    Use the the 'IR Code available' instruction to check the availability of a IR code and then, if the code number is smaller than 9 then print the message.

     
  3. IR Channels II

    Now, improve the previous exercise to also print a message when the power off button is pressed (code 12 in our case).

    Hint:

    Now, you need to store the code on a variable and extend the 'if' instruction to check if the code number is smaller than 9 to print the channel message or else if it is equal to 12 to print a power off message.

Intermediate Exercises
  1. IR 'Piano'

    Generate a music tone based on the infrared code received to create a remotely controlled piano.

    Hint:

    Use the 'Advanced Buzzer' instruction to conver the code number into a half-tone distance. You can set as initial frequency 220Hz.

  2. IR Commands

    Set the color of an RGB LED based on the received infrared code. If you press the power off button, the RGB LED must be off, while if you press '1', '2' or '3', you must set the RGB LED red, green and blue, respectively.

    Hint:

    Use the 'IR Command' instruction to create four cases, with codes '12', '1', '2' and '3'. On each case, use the 'RGB LED' instruction to set the corresponding color.
