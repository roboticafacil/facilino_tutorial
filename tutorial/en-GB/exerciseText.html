Text

Learn how to search, crop, append and compare text with Facilino.

Required material.
  • An Arduino or ESP board, i.e.: Arduino Uno, Arduino Nano, Wemos D1 R2 or Wemos D1 R32, NodeMCU, etc...
  • A Multisensor shield or two LEDs (red and blue connected to pins D12 and D13), a potentiometer connected to pin A0 and two pushbuttons connected to pins D2 and D3.
Worked instructions
Basic Exercises
  1. Concatenate Text

    Use the 'Create text with' instruction to concatenate text with numbers to print the value of the potentiometer connected to pin A0 with a message before the value, i.e.: 'The value of the potentiometer is '.

    Hint:

    Remember to include a space at the end of the text so that the text and the number are separated.

     
  2. Spell it out

    Spell out any word read from the serial monitor. To spell out a word, you must use the tab.

    Hint:

    Use the instruction to get the character at the indicated position. With a 'While' loop, get read all characters from the string and print them (without a line break, it does not actually print, it just simply sends characters to the serial monitor, but nothing will be shown untile the line break character is received). Once the word has been spelled out, you can use the print line instruction to actually print it back on the serial monitor.

     
  3. Lower and upper case

    Read a string from the serial monitor. If pushbutton connected to pin D2 is LOW, then print the text in lower case, if the pushbutton connected to pin D3 is LOW, then print the text in upper case. Otherwise, print back the text as it is.

    Hint:

    Use 'lower' and 'upper' case instructions to modify the text.

Intermediate Exercises
  1. Reverse string

    Create a program that reads a string message over the serial port and prints the reserved message.

    Hint:

    First read a string message over the serial port using the 'read string' instruction (check data availability on serial port). Afterwards, in a for loop, iterate over each of the characters of the message in reverse order (from n-1 to 0, being n the length of the string message) and append each of the characters to a string variable, so that after the for loop, you have the string reversed.

     
  2. Number of words

    Create a program that reads a string message over the serial port and counts the number of separated words (prints a message "Number of words: xxx", being xxx the number of words).

    Hint:

    Implement a while loop and check if the message length is greater than 0. In the while loop, count the number of times a space character is found and (store the result in a variable "new_pos" and check if the difference with respect to the previous position "pos" is greater than zero). On each iteration of the while loop, crop the string message from "new_pos+1" to "n-1", being "n" the message length. To print the message with the number of words, use the instruction 'create text with' so you can concatenate a string and the result of the word count.

     
  3. Searching text

    Create a program that makes the red LED (connected to pin D12) blink every time the user sends a message with the word 'Facilino' and makes the blue LED (connected to pin D13) blink every time the message contains the word 'Arduino'.

    Hint:

    Use the 'contains...expression' instruction in the text category.
