Text

Learn how to search, crop, append and compare text with Facilino.

All required Electronics are already integrated in the Multisensor Shield.
Required material.
  • Arduino Uno (and compatible) or Wemos D1R2/D1R32.
  • A Multisensor Shield.
Worked instructions
Exercises
  1. Reverse string

    Create a program that reads a string message over the serial port and prints the reserved message.

    Hint:

    First read a string message over the serial port using the 'read string' instruction (check data availability on serial port). Afterwards, in a for loop, iterate over each of the characters of the message in reverse order (from n-1 to 0, being n the length of the string message) and append each of the characters to a string variable, so that after the for loop, you have the string reversed.

  2. Number of words

    Create a program that reads a string message over the serial port and counts the number of separated words (prints a message "Number of words: xxx", being xxx the number of words).

    Hint:

    Implement a while loop and check if the message length is greater than 0. In the while loop, count the number of times a space character is found and (store the result in a variable "new_pos" and check if the difference with respect to the previous position "pos" is greater than zero). On each iteration of the while loop, crop the string message from "new_pos+1" to "n-1", being "n" the message length. To print the message with the number of words, use the instruction 'create text with' so you can concatenate a string and the result of the word count.

  3. Searching text

    Create a program that makes the red LED (connected to pin D12) blink every time the user sends a message with the word 'Facilino' and makes the blue LED (connected to pin D13) blink every time the message contains the word 'Arduino'.

    Hint:

    Use the 'contains...expression' instruction in the text category.
