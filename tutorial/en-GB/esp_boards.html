ESP Microcontrollers

ESP boards are lower-cost and higher performance than competing microcontrollers such as Arduino. If your never heard of them before, is because these boards are relatively new, compared to Arduino boards. At first glance getting one of these configured can be intimidating but with all the tools and resources available today this is actually a straight forward process (see Installation and Setup Section). In the end, you’ll be glad you invested in time setting up your ESP board as they are much more equipped for the price are fully ready for IoT applications.

The ESP is a microchip with full TCP/IP stack and produced by manufacturer Espressif Systems in Shanghai, China, a semiconductor company that just came out of nowhere and shook up the whole industry. The chip first came to the attention of western makers in August 2014 with the ESP-01 module (based on ESP8266 processor), made by a third-party manufacturer Ai-Thinker. ESP-01 came with a simple serial interface based on AT commands used in modems. Because of its simplicity and price became very popular within makers community using Arduino to extend WiFi connectivity to Arduino Uno. In 2015, Espressif released the API to be able to program their ESP8266 microcontroller.

Also in 2015, a company-independent group of people develop a LUA parser based on ESP8266 and their own board, known as NodeMCU. This board can be integrated in Arduino IDE, because there's an API mantained by the ESP8266 Community and a tool, developed by Ivan Grokhotkov, known as esptool. The API with libraries and tools is maintained by the community on GitHub.

The ESP32 processor came out at the end of 2015 and overcomes some limitations on ESP8266 boards. It is much faster. It has two cores and 520Kb RAM shared by the cores and WiFi encryption is carried out by dedicated hardware. ESP32 includes: 2.4 GHz WiFi, Bluetooth 4.2, dual high performance core processor, 520KiB SRAM, 160-240MHz, 2 x 8-bit DAC’s, 12-bit SAR ADC up to 18 channels, I2C interfaces, 3 x UART. While ESP8266 includes: 2.4 GHz Wifi, 32-bit RISC, 80MHz, 16 GPIO pins, I2C interfaces, 32KiB instruction RAM, SPI, I2S, 10-bit ADC.

The API to program ESP32 boards in Arduino IDE is mantained by Espressif Systems Company on GitHub.

Wemos Boards

Wemos is a Company that develops boards using ESP processors. There are two boards that are worth mentioning: Wemos D1 R2 and Wemos D1 R32. The main reason to focus on these boards is because they are compatible with Arduino Uno boards, which means that the pins layout is the same as Arduino Uno. Despite of the similarities there are obviously differences between them, because pins can have multiple purposes. For instance, Wemos D1 R2 has only one analog input and 9 digital I/O pins, while in Wemos D1 R32, pin names are different than conventional Arduino Uno naming.
