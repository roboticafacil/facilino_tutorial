Facilino Installation & Setup

Install Facilino before you start programming.

Installing Facilino

You can download Facilino from Robotica Facil website or work in a browser-based version and Arduino IDE. Note while the downloaded version requires a local installation it has significant more processing power. This is highly recommended as the next instructions will follow that version and you can directly upload your code using Facilino (Arduino IDE is required to be installed so that your programms can compile and upload). If you decide to use the browser-based version, please follow the instructions refered to in the Facilino browser-based version section at the bottom of the page. Below you will find a step by step instruction for setting up the downloaded version for Microsoft Windows. For Ubuntu, please follow Ubuntu Installation Instructions

 

  1. Download Files

    First, download Facilino from the Robotica Facil website.

    Select your computer specific version and press the link to start downloading. Be advised that these downloads are taken from Robotica Facil's GitHub, where you can find the source code of the application. 

  2. Execute Files

    Click on the downloaded file to execute it. Your computer might give a warning as the Installation comes from an unknown source. Simply, give permision under the extended tab option. The Facilino software is complety safe and does not pose a threat to your computer.

  3. Facilino Setup

    The Facilino Setup should open, make sure to select the Boxes; Facilino, Arduino Libraries and Start Menu. If you do not have already a Arduino IDE program installed also Select Arduino IDE. Facilino is the actual application and it is required. Arduino IDE is the Arduino Software Facilino integrates in and it is highly recommended for new installations. Arduino/Facilino libraries are the additional libraries needed by some Facilino blocks. The setup directly installs them in the Arduino IDE and may overwrite older Versions in your current Arduino IDE to asure compatibility. They can be changed and modified again later. Start Menu and Deskop Icon create the mentioned Start Menu and Deskop Icon for easier access from your home screen.

  4. Facilino Installation Location

    You know have to select the installation location, this is automatically set to your folder C:\Facilino. We recommend you not to change this path to avoid possible future administrative permissions. 

  5. Facilino Installation Process

    After confirming, your installation should start immediately and a progress bar should be visible. This installation might take a few minutes, particularly when installing Arduino/Facilino libraries, because they will be first downloaded and then installed.

  6. Facilino

    Select the Facilino icon from the Desktop or the Start Menu. The Facilino should start up. You need only a couple more steps before you can start creating code. On the Top Menu select Preferences and insert the License (if available) in the dialog window. Reasure that you have the right Arduino IDE path selected (in Windows this should point to arduino_debug.exe file. Close the window and click on Reload.

    At the bottom-right corner you have to select your microcontroller type and selected USB Port (with the board connected to the USB port).

  7. Upload

    With the top-right corner icons you can Verify and Upload your code. You can try a blank project with just the 'Setup/Loop' instruction to make sure everything is working order and no error messages are beeing displayed. If that is the case, read the Additional Help section at the bottom of the page.

    If everything works correctly you should have the following messages in the information window that appears when verifying or uploading the code.

    For Instructions on how to use Facilino and work with the different kind of blocks please check the Facilino Section.

Facilino Browser-based Version

Facilino should run normally in Windows, Ubuntu and macOS (Mojave). However, there might be some installation issue that might cause problems during the execution of the application. For that reason, we developed the browser-based version, so-called online version. Facilino is a light-weight web application which makes easy to use and works in Firefox, Chrome and Safari. Access to Facilino browser-based version in Robotica Facil website

The main difference with respect to Windows, Ubuntu or macOS version is that the browser-based version is an Arduino code generator, which means that you will need to copy the code in Arduino IDE and upload the generated code from there. In order to successfully upload your code, you will need to have all required libraries installed in Arduino IDE, as well as board manager settings. Otherwise, you will get compilation errors. Each Facilino instruction is documented and the dependencies with external libraries is usually indicated on the instruction documentation.

Troubleshooting
  1. Code errors Incorrect use of a block might generate an compilation error, due to incorrect generated code. Facilino checks your code and in most of the cases this prevents for errors or generates warnings at potentially incorrect use of the instruction. Despite of that, there are situations that will still generate compilation errors because your code is incomplete or not used as expected. For instance, this code will generate an error due to missing inputs indicating the pins where the electronic is connected to.
  2. Missing libraries Some blocks required external libraries to properly work. Please read the documentation on each instruction to succeed compiling your code. The Windows version of Facilino installs required libraries, so this issue should not appear. However, other versions of Facilino do not cover this feature, so it is your task to install required libraries in Arduino IDE.
  3. Arduino Compiler Facilino uses Arduino compiler to successfully verify and compile your code. The path to the compiler must be properly set in the Preferences of Facilino. In Windows this should point to arduino_debug.exe file. In Ubuntu, the path for the compiler is usually installed in /usr/bin/arduino, while in macOS should point to Arduino.App.
  4. License This is one of the most common issues if you don't know how Facilino works. First, Facilino is a freemium program, which means that many of its features are for free. Advanced instructions will require a license, a 13-digit digital key that you can get at Robotica Facil website. Without the license, some blocks won't generate the corresponding Arduino code, which will likely generate compilation errors or simply not working as expected. There are two types of licenses: individual and server. The individual license expires after one year of use, it is intended to be used of personal use and after one year. The server license, will allow you to use the license in multiple machines and it does not expire. Please, refer to the license conditions in Facilino license.

    You can set your license in the Preferences of the application. At start-up the application will check the validity of your license. Sometimes this process doesn't work. If that's the case, check your firewall settings (there must be a rule allowing traffic Facilino app). In most of the cases, the issue arises due to slow internet connections. You can use the Reload button at the toolbar to re-validate your license.
