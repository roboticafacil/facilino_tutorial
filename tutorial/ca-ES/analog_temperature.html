Sensor de Temperatura Analógico

Mide temperatura con precisión elevada.

Sensor LM35

El LM35 es un sensor de temperatura analógico. Tiene una respuesta lineal con una resolución de 10mV por grado Centígrado. Alrededor de 25 ° C tiene la precisión más alta, con una precisión de 0.5 ° C, su rango es de -55 ° C a 150 ° C. Tiene tres pines, uno conectado a tierra, uno conectado a la fuente de alimentación de 5 V y el último para la señal analógica. El sensor LM35 se puede encontrar como uno de los sensores incluidos en la placa multisensor

El módulo del sensor de temperatura KY-070 permite comenzar a practicar con este sensor de forma sencilla. Puedes conectar su señal analógica a una entrada analógica del microcontrolador para medir la temperatura.

Codificando Melodias Con Facilino

Facilino incluye una instrucción para leer la temperatura de un sensor LM35. La instrucción acepta como entrada el número de pin analógico y devuelve la temperatura en grados Centígrados.
